### Introduction
This is a project that allows you to play a liar game using generative AI images. <br/>
This project was created for submission to a departmental promotional project competition. <br/>
It was developed with the idea of how to make artificial intelligence engaging and accessible for middle school students.

### Key Features
- AI image generation functionality
- Basic liar game features
  - Topic selection
  - Liar selection, etc.
